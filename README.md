# stm32-base-project

### Visual Studio Code環境設定
以下のエクステンションをインストールする。
```console
$ code --install-extension　marus25.cortex-debug
```

### armコンパイラのインストール
```console
$ sudo apt install arm-none-eabi-gcc
```

### ビルド
```console
$ ./build.sh
```
